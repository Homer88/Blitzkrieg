# Структура проекта Blitzkrieg

## Описание

Blitzkrieg — это общий проект сборки, состоящий из множества модулей. Каждая папка в директории `src` представляет собой отдельный проект (библиотеку или исполняемый файл), которые вместе образуют единое решение `Blitzkrieg.sln` в Visual Studio.

## Структура модулей

## SDK

Проект использует следующие SDK:

- **DirectX 8** (`../sdk/directx/`) - основной графический API
  - Include: `../sdk/directx/include/`
  - Libraries: `../sdk/directx/lib/`

## Сборка

### Быстрая конфигурация

```batch
# Для x86 (32-bit)
configure_x86.bat

# Для x64 (64-bit)
configure_x64.bat

# Интерактивный выбор версии Visual Studio
generate_vs_project.bat
```

### Ручная сборка

```batch
mkdir build
cd build
cmake -G "Visual Studio 17 2022" -A Win32 ..
```

### Результат сборки

После выполнения скрипта конфигурации в папке `build_x86` (или `build_x64`) будет создан файл решения:
- `Blitzkrieg.sln` - общий файл решения Visual Studio

Каждый модуль будет отображаться как отдельный проект в решении.

## Совместимость

- **Стандарт C++**: C++98
- **Минимальная ОС**: Windows 98 / Windows 2000
- **Компилятор**: MSVC (Visual Studio 2013+)
- **Архитектура**: x86 (32-bit), x64 (64-bit)

## Примечания

- Все проекты используют стандарт C++98 для максимальной совместимости
- Для Windows 98/2000 установлен `_WIN32_WINNT=0x0400`
- DirectX 8 SDK подключается глобально ко всем проектам
