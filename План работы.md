## Примечания по исправлениям

### AILogic (было 600 ошибок, осталось 12)
**Исправлено:**
- Создан `std_hash_compat.h` — адаптер `std::hash_map` → `stdext::hash_map` для MSVC 2022
- Исправлены `DTHelper.h`, `SSHelper.h`, `RPGStats.h` — добавлен `std_hash_compat.h`
- Создан `AILogicMissingTypes.h` — `IGunDescriptor`, `SUpdatableObjectPtr`, `SUnitRPGInfo`, `SDiplomacyInfo`
- Исправлен `Formation.h` — добавлен missing return type `const bool`
- Исправлен `UnitsIterators2.h` — `template<BYTE false, ...>` → `template<BYTE cOnlyVisible, ...>`
- Исправлен `UpdatableObject.h` — `CTilesSet` определён ДО `RectTiles.h`
- Исправлен `Commands.h` — добавлен `#include "ListsSet.h"`
- Добавлены заголовки в `StdAfx.h`: `AIGeometry.h`, `AIConsts.h`, `AIInternalConsts.h`, `Commands.h`, `ListsSet.h`, `AIUnit.h`, `UpdatableObject.h`, `Units.h`, `UnitsIterators2.h`, `Guns.h`
- Добавлен `SVector::operator<` в `AIGeometry.h`

**Оставшаяся проблема — circular include:**
```
AIHashFuncs.h → AIUnit.h → CommonUnit.h → LinkObject.h → UpdatableObject.h → RectTiles.h → AIHashFuncs.h
```
Когда `AIHashFuncs.h` обрабатывается через этот путь, `CAIUnit` ещё не определён.
**Решение**: добавить `#include "StdAfx.h"` во все 169 .cpp файлов (сейчас только 10).
